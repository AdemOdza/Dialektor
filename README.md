# Dialektor

A website to show off how words are said/spelled in different dialects of the Albanian language

## Project Structure

- `dialektor-backend/` - Python/Flask backend API
- `dialektor-frontend/` - React.js frontend application

## Frontend

The frontend is built with React.js and Vite. See [dialektor-frontend/README.md](dialektor-frontend/README.md) for details.

### Quick Start (Frontend)

```bash
cd dialektor-frontend
npm install
npm run dev
```

## Backend

### Commands
- Install dev dependencies: `poetry install --with dev`
- Format code: `poetry run black ./src`
